---Versión ReactJS---
Para probar la versión ReactJS del proyecto Tienda:

1) Acceder a la carpeta del proyecto, Tienda_ReactJS, mediante la consola de comandos.
2) Ejecutar "npm install" para descargar todos las dependencias npm requeridas.
3) Aparte, ejecutar un servidor Mongo con mongod (no incluído en el proyecto) y dejarlo andando.
2) De nuevo en la carpeta del proyecto, ejecutar "npm run generarbd" para crear una base de datos con 3 usuarios y los productos de la tienda con stock al azar.
3) Ejecutar "npm start" para iniciar webpack-dev-server, dejarlo andando.
4) Ejecutar "npm run servidor" para iniciar el servidor de la tienda, dejarlo andando.
5) Ir a http://localhost:8080 en el navegador.
